屍人荘の殺人

BOOKS
发布时间 : 2024-02-19 21:40
阅读 :

推理小说，被读者猜到谜底比什么都糟糕。制造悬念的环节变成等待侦探察觉真相的焦躁，真相的揭示不再有出乎意料的惊喜，让故事高潮的剧情和读者的情绪脱节，一些夸张的演出会看起来滑稽可笑。我宁可看到作者给我强行捏造一个反转的结局，也不想出现这种直球谜题。

这本书最大的问题就在这。先从why done it 这一方面开始，连一点点掩饰都不做，把最明显的动机从故事最初就反复强调，也没想到最后真的没有任何动机方面的反转。而且还是最典型的模板式动机加人设，谁是受害者谁是杀手一目了然。虽然这些都是读者在剧本外的超游思维，但写书的目的是取悦读者，你不能指望读者一直跟着故事内的讯息走。这书写成的时候都快20年了，真的没必要搞这种无聊的男女关系复仇剧。死者是公子哥玩弄感情的渣男，杀手是人群中表面最安静的女孩子。典型到我都怀疑是作者直钩钓鱼的程度。这个背景设定和人设也是，确实是二次元推理作品的风格，套的模板太老土了。看出了一种gal或者轻小说里那种强行插入福利情节的生硬感。

How done it 的叙述也很直接，比如说刻意点出电梯限重和插电钥匙卡不能替换的这些信息，相当于提示了作案手法。后面往这些方面靠很容易就能自己判断出作案手段。把这些信息换一种委婉的方式点出会更好。比如说电梯这里直接安排一个情节，就是大家一块上电梯之后电梯响了不能运作，一个人退出去之后又好了。进而通过在场人员体重让读者推算出电梯限重不就好了。本书就直接让管理员开口说电梯限重两人，这种常识性问题让角色亲口说出来不觉得很突兀吗，直接就和关键线索挂上钩，让读者意识到后面推理绝对会用到这一条线索。

这本书的谜题构造是可以的，但是叙述方法的提升空间真的很大。

这本书唯一的优点可能只有他详细的介绍了非法潜入房间的方法了。这个豆知识还是挺有趣的

开酒店门需要一个铁丝弯钩，从门缝里伸进去从内侧划开门锁的锁头位置。

开铁链则需要用胶带和橡皮筋，用橡皮筋挂住铁链并用胶带把橡皮筋尽量固定到门框的高处，这样关闭门的时候铁链就会被皮筋提起来。所有像一个方向移动就能打开的挂锁装置都能用这个方法打开。

烂书，这种作品都能得奖，我很怀疑现在日本这么多推理小说到底有多少是能看的。日本书店这么多书恐怕都是网文水平吧。
